ByeDPI Web Manager (для Windows)

Кратко:
- Генератор стратегий.
- Проверка стратегий через указанные или выбранные ссылки.
- Управление процессами ByeDPI для дальнейшего постоянного использования.

Требования:
- PHP версия 8.0.0 и выше.
- Включённые PHP расширения:
  - extension=curl
  - extension=openssl
  - extension=com_dotnet
- Актуальный curl_cert\cacert.pem взятый из https://curl.se/ca/cacert.pem

Настройка:
- config.json
  - Настройка IP для запуска.
  - Настройка IP для PAC файла и отображения в скрипте.
  - Настройка портов (если они заняты другими приложениями).
  - Настройка групп ссылок по умолчанию.

Функционал:
- Автоматическая проверка системных требований и утилит для работы: система, PHP, утилиты, файлы.
- Попытка автоматического применения необходимых прав к файлам.
- Определение текущих серверов текущего кластера Google Global Cache.
- Проверка портов и запущенных процессов ciadpi.exe.
- Процессы "для проверки стратегий" будут автоматически завершены при запуске скрипта (если вдруг что-то пошло не так).
- Процессы "для использования" при перезапуске скрипта завершены не будут (будет определён их статус: запущен или не запущен).
- Генерация стратегий на основе весовых коэффициентов для аргументов и флагов.
- Настройка php_curl запросов.
- Многопоточная проверка: чем больше ссылок, тем дольше ждать первый результат.
- Фильтрация проверок по проценту успешных запросов.
- Использование списков хостов.
- ByeDPI 0.17 (x86_64) в комплекте.

Подключение к процессам "для использования":
- Socks5
- PAC

пы-сы: 
Для ПК: При применении socks5 или PAC не забываем "не спеша" перезагружать браузер, чтоб "подтянулись" новые настройки. 
Для телефона/tv: При применении socks5 или PAC не забываем "отключить wifi", затем "включить wi-fi", чтоб "подтянулись" новые настройки.

Определенные Ваши текущие GGC сервера будут в блоке "Ссылки для проверки стратегий" в селекте. Если не удалось определить или что-то пошло не по плану с этим пунктом, оно не является критическим, на работу скрипта не повлияет.
